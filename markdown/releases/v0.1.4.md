# v0.1.4 - Global Glossary Admin Foundation

Release date: 2026-06-26

Summary: Adds the Admin-managed global glossary foundation used to define approved terms and aliases for later participant inline definitions.

## Added

- Admins can manage global glossary entries with a canonical term, plain-text definition, match strings/aliases, and enabled or disabled state.
- Glossary entries can be created, edited, enabled, disabled, and archived from the Admin tools area.
- The glossary data model stores dictionary-assist source metadata for a future suggestion workflow.

## Security

- Glossary management APIs and pages are Admin-only, and participant-facing survey behavior remains unchanged.

## Operational Notes

- Adds a database migration for global glossary entries and match strings.
