# v0.1.4 - Builder Workspace And Mobile Polish

Release date: 2026-06-12

Summary: Splits the Admin builder into a more usable workspace and adds mobile-first visual polish across core Admin and participant surfaces.

## Added

- Survey workspace tabs for Setup, Questions, Organize, Logic, Preview, and Results using the existing survey data contract.
- Responsive builder layout, mobile navigation controls, stronger focus-visible outlines, and improved action stacking for narrow screens.
- Early Admin information architecture that separates survey definition work from survey review and reporting work.

## Changed

- Builder pages now emphasize operational workflows instead of one long mixed editing screen.
- Mobile and keyboard interaction patterns were tightened without changing survey data contracts.

## Security

- No new backend authorization surface was added; the workspace continues to rely on the Admin guard and server-side Admin checks.
