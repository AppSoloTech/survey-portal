# v0.4.0 - Responses, Timing, And Tag Foundations

Release date: 2026-06-24

Summary: Expands answer capture, hidden-tag modeling, completion-time estimates, participant timing display, activity instrumentation, and tag catalog grouping.

## Added

- Choice-based questions can enable a system-generated Other response with separately stored free text and reporting/CSV support.
- Admins can attach hidden tags to Other responses while keeping Other out of normal answer options and conditional logic.
- Survey-level completion-time estimates derive from completed attempts, fall back to backend question-type weights, and can be overridden by Admins.
- Participant runners show estimated time remaining from the effective survey estimate instead of numeric progress labels.
- Authenticated and anonymous attempts write lightweight activity events for future dynamic timing improvements without exposing timing internals to participants.
- Tag catalog groups let Admins organize hidden-tag vocabulary and keep builder-created tags aligned with the catalog.

## Changed

- Profile demographic fields were cleaned up into structured address and phone metadata, with Settings and Admin user detail updated to show the active shape.
- Tag terminology now uses category/value language consistently across Admin validation, catalog copy, and results helper text.
