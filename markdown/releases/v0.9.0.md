# v0.9.0 - V1 Readiness Roadmap

Release date: 2026-06-29

Summary: Consolidates the app roadmap to the current V1-readiness line and adds Admin review tagging for qualitative open-text survey responses.

## Added

- Admin Results can apply and remove catalog-based manual review tags on individual text-question answers.
- Review-tag backend stores catalog/tag-definition references, exposes Admin-only add/remove endpoints, and includes review tags in attempt detail.
- CSV export now separates manual review tags into a dedicated `review_tags` column instead of mixing them with automatic hidden tags.
- The Admin Software updates page now reflects a milestone roadmap built from the phase log and prompt history through the current `0.9.x` target.

## Changed

- Release history has been reorganized from sparse patch notes into readable product milestones with phase-source notes for each entry.
- The root app version now tracks the consolidated V1-readiness roadmap as `0.9.0`.

## Security

- Review-tag routes remain Admin-only and validate answer ownership, text-answer eligibility, catalog membership, and idempotent add/remove behavior server-side.

## Operational Notes

- Roadmap source: Phases 47-48 plus this release consolidation pass, using `markdown/PHASE_LOG.md`, `prompts/prompt_47.txt`, and `prompts/prompt_48.txt`.
- Remaining V1 follow-ups include whether Other text, integer answers, or review-tag aggregate rollups should receive similar manual coding support.
