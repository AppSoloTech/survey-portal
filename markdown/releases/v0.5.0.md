# v0.5.0 - Public Directory, Conversion, And Release Tracking

Release date: 2026-06-25

Summary: Adds public anonymous discovery, anonymous-to-account conversion, and the Admin-visible software update workflow that powers this roadmap.

## Added

- Public anonymous survey directory lists only Admin-approved, published, enabled, unexpired anonymous survey links.
- Admin Setup controls can opt individual anonymous links into or out of the public directory.
- Anonymous participants can create a registered account after completing a survey and attach the completed anonymous attempt to the new account when eligible.
- Admin-only Software updates page reads committed versioned Markdown release notes and displays the current root app version.
- Release-note CLI scripts scaffold, preview, promote, validate, and test release notes as part of deployment preparation.

## Changed

- Deployment and direct-main workflows validate release notes before production pushes so the Admin page stays aligned with version control.

## Security

- Public directory responses expose only participant-safe survey metadata, and anonymous conversion rechecks link availability and ownership before account attachment.

## Operational Notes

- Roadmap source: Phases 31-33 in `markdown/PHASE_LOG.md` and `prompts/prompt_31.txt` through `prompts/prompt_33.txt`.
- Release files under `markdown/releases/` are the source of truth for the Admin roadmap page.
