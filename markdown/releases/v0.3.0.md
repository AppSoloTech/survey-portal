# v0.3.0 - Page-Based Survey Flow

Release date: 2026-06-17

Summary: Reworks surveys around pages, adds page-aware organization and logic, and corrects branching behavior so stored answers match the respondent's final path.

## Added

- Page model, page CRUD/reorder endpoints, page-scoped question ordering, question movement, page-aware duplication, and page-aware rule validation.
- Participant runner support for rendering one page at a time and saving page answers in batches.
- Organize tab and page-focused Questions tab so Admins can structure surveys without mixing page layout and question editing.
- First-class `HIDE_PAGE` skip-page rules with optional advance-on-trigger behavior and flow-map descriptions.

## Fixed

- Off-path answers are pruned at save time when participants change a branching answer, preventing abandoned-branch answers from inflating aggregate reports or hidden-tag counts.

## Changed

- Reporting and path resolution now align more closely with the page runtime for jump, skip, hidden, and final-path state.
