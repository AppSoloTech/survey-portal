# v0.9.2 - Capacity Suites And Assessment UI Polish

Release date: 2026-06-30

Summary: Adds the next capacity-testing foundation while polishing user-facing assessment language and question rendering behavior.

## Added

- Add the read-only Admin API and schema foundation for performance capacity suites, child-run relationships, and bounded time-bucketed sample evidence.
- Add the operator-run capacity suite CLI with small, standard, and opt-in capacity presets, linked child run persistence, bounded local k6/SQL samples, aggregate reports, and conservative bottleneck classification.

## Changed

- Rename user-facing "survey" copy to "assessment" across participant, public-directory, account, and Admin workspace UI surfaces while preserving existing routes, APIs, and stored data contracts.

## Fixed

- Fix glossary definitions staying visible when tooltip popovers should be hidden after the accessibility tooltip update.
- Stabilize Safari's initial question prompt layout so question text no longer briefly stacks vertically before refresh or resize.
