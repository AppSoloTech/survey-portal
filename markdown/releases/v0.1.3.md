# v0.1.3 - Admin Builder Maintenance

Release date: 2026-06-26

Summary: Improves admin survey-builder maintenance workflows, including live hidden-tag corrections, tag catalog organization, and focused question editing.

## Added

- Admins can add, edit, and remove hidden tags on published surveys while questions, options, pages, and rules remain locked.
- Text and integer value tags can now be edited directly instead of requiring delete and recreate.
- The Ungrouped tag catalog section can now be reordered with custom tag categories and keeps its position after reload.
- Question editors now group detailed tooling into collapsible sections so admins can focus on one part of a question at a time.

## Changed

- Reports, attempt detail, and CSV exports reflect current hidden tag metadata for already answered published surveys.
- Tag catalog copy now uses category language for higher-level catalog buckets.
- Survey workspace tabs on narrow screens now show a horizontal scroll affordance when additional tabs are off-screen.
- The Questions page Editing page selector now stacks cleanly on narrow screens.

## Security

- Hidden tags remain admin-only and are still excluded from participant survey payloads.
