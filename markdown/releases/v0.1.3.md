# v0.1.3 - Admin Builder MVP

Release date: 2026-06-11

Summary: Adds the Admin survey builder, early builder polish, and visual flow mapping for creating, publishing, and understanding survey logic.

## Added

- Admin builder APIs and UI for survey setup, question creation, answer options, hidden tags, conditional rules, publish, retire, and republish workflows.
- Publish validation for required questions, option-backed question integrity, and same-survey conditional rule targets.
- Scoped mutation feedback, clearer status action panels, safer destructive labels, and builder copy for jump-target behavior.
- Survey flow map visualization with accessible nodes, normal paths, conditional edges, incoming path notes, legends, and issue summaries.

## Changed

- Forward-only jump validation and saved-response protections keep draft edits from deleting data that participants have already answered.
- Jump targets can be excluded from normal linear flow, with Admin control over whether conditional targets stay in the default path.

## Security

- Builder and publishing routes are Admin-only, and hidden-tag metadata remains excluded from participant-facing survey responses.
