# v0.3.1 - Communication, Anonymous Access, And Profiles

Release date: 2026-06-22

Summary: Adds the email abstraction, anonymous survey foundation, password reset workflow, user profile tools, and Admin user management foundation.

## Added

- Provider-agnostic email client foundation with disabled and development no-op adapters, safe environment defaults, and sparse non-sensitive logging.
- Anonymous survey links with hashed secrets, encrypted Admin-copyable public tokens, per-attempt anonymous access tokens, PostgreSQL-backed public-route rate limiting, and reusable participant runner support.
- Password reset request and completion flows from login and account settings, using reset tokens, rate limiting, generic responses, and the email abstraction.
- User Settings profile foundation with contact fields, survey stats, editable profile metadata, and Admin user detail views with password reset initiation.

## Changed

- Admin reporting labels anonymous attempts separately from registered users and surfaces optional anonymous follow-up email addresses as unverified contact data.

## Security

- Anonymous routes remain cookie-free, validate link availability on every read/write, avoid hidden-tag exposure, mask public tokens in logs, and use `Referrer-Policy: no-referrer`.
- Password reset avoids account enumeration, stores token secrets as hashes, and keeps production email delivery disabled until a real provider is approved.
