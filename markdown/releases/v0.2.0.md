Release date: 2026-06-16

Summary: Adds Admin reporting, CSV export, automated tests, migration tooling, CI, hosted deployment checks, and early production hardening.

- Login and password guardrails, production proxy trust, health failure handling, request logging hygiene, and deployment preflight checks harden the operational baseline.
