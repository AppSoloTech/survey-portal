# v0.1.0 - Public Pilot Baseline

Release date: 2026-06-24

Summary: Baseline release for the Survey Portal MVP public pilot, including the core survey workspace, admin tools, anonymous survey flow, reporting, and security hardening foundation.

## Added

- Core React and Express survey portal with registered user dashboards, admin survey management, and PostgreSQL-backed data storage.
- Admin survey builder covering setup, questions, organization, conditional logic, preview, results, users, tags, and categories.
- Anonymous survey directory and token-based anonymous survey attempts.
- Phase workflow documentation under `markdown/` and implementation/review artifacts under `notes/`.

## Security

- Cookie auth, admin role checks, CSRF protection for unsafe browser requests, security headers, request redaction, and session-version invalidation.
- Hosted deployment workflow with lint, build, test, audit, secret scan, and Azure deploy steps.

## Operational Notes

- Database migrations are managed through ordered SQL files in `database/migrations/`.
- Hosted releases ship through `git push origin main` or `npm run deploy`.
