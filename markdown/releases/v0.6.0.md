# v0.6.0 - Templates, Builder Maintenance, And Glossary Admin

Release date: 2026-06-26

Summary: Adds reusable Admin templates, safer published-survey maintenance, improved tag catalog controls, focused question editing, and the global glossary Admin foundation.

## Added

- Page templates let Admins save and insert reusable page structures into draft surveys with copied questions, options, scale values, hidden tags, and Other hidden tags.
- Question templates let Admins save and insert reusable question definitions into draft survey pages.
- Combined template library supports page/question kind labels, search/filter, source metadata, rename, delete, detail review, and draft-only insertion placement.
- Global glossary Admin page and APIs let Admins manage canonical terms, aliases, plain-text definitions, enabled state, and archive behavior.

## Changed

- Published surveys can receive hidden-tag maintenance without unlocking questions, options, pages, or rules.
- Text and integer value tags can be edited directly, and reports/attempt detail/CSV export reflect current hidden-tag metadata.
- Tag catalog section ordering includes Ungrouped and custom categories, and question editors group detailed tooling into collapsible sections.

## Security

- Template, published-maintenance, tag catalog, and glossary management remain Admin-only; participant-facing survey payloads do not include hidden-tag or glossary source metadata.
