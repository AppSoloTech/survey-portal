# v1.0.0 - Participant issue profile thermometer and emoji burst

Release date: 2026-06-30

Summary: Adds a participant-facing issue profile thermometer and emoji burst that make assessment progress feel more responsive while preserving hidden-tag privacy.

## Added

- Added an abstract issue profile thermometer to registered and anonymous assessment attempts.
- Added participant-safe aggregate issue profile progress to runner responses.
- Kept the issue profile thermometer visible in a sticky runner header, with a ready-to-submit completion display before final submission.
- Added nullable emoji metadata to admin tag catalog entries.
- Added an admin tag-category emoji action so existing hidden tag categories
  can receive one emoji across all of their catalog values.
- Added participant-safe issue profile emoji aggregates to registered and anonymous runner responses.
- Added a Ready to submit emoji burst from the issue profile thermometer with reduced-motion fallback.

## Changed

- Refined the Ready to submit emoji burst so it can replay when returning to
  the review screen, uses a slower upward burst and falling arc, and includes
  decorative spark particles.

## Security

- Preserved hidden-tag isolation for participant and anonymous payloads while computing thermometer state server-side.
- Preserved hidden-tag isolation for the emoji burst by exposing only emoji/count aggregates to participants.
