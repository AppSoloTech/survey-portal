# Codex Handoff — Phase 47 Open-Answer Review Tagging Backend Foundation

## Scope Implemented

- Added `database/migrations/0036_response_answer_tags.sql`.
- Added `prompts/prompt_47.txt`.
- Added shared Admin review-tag types and `responseAnswerId`/`reviewTags` on
  `AdminAttemptAnswer`.
- Added Admin-only add/remove review-tag routes under survey reporting.
- Added text-only server validation for saved nonblank `text` response answers.
- Added review tags to Admin attempt detail.
- Added separate `review_tags` CSV export column.
- Added backend reporting tests for auth, ownership, text-only validation,
  duplicate add, remove, cascade cleanup, attempt detail, and CSV.

## Important Decisions

- V1 supports only dedicated text-question answers.
- Integer answers stay covered by automatic value/range tags.
- System-generated Other text is deferred.
- Review tags reference `tag_definitions.id` instead of copying key/value text.
- Review tags export separately from automatic `hidden_tags`.
- Review-tag aggregate rollups are deferred.
