Key implementation decisions:
- The canonical term is persisted as the first glossary_match_strings row with
  is_canonical = true. This makes the active case-insensitive unique index on
  normalized_match_text cover canonical terms and aliases uniformly.
- DELETE archives by setting deleted_at on both glossary_entries and active
  glossary_match_strings. Archived match strings are excluded from the active
  unique index and can be reused later.
- Phase 39 stores dictionary-assist metadata fields but does not call any
  external dictionary provider.
- Participant-facing survey APIs and runner behavior were intentionally left
  unchanged.
