# Unreleased

Release title: Admin Page Templates

Summary: Adds reusable admin page templates so admins can save common survey pages and insert them into draft surveys.

## Added

- Admins can save a survey page as a reusable page template and insert it into another draft survey.
- Page templates copy page content, questions, options, scale ranges, and admin-only hidden-tag metadata.

## Security

- Page template APIs are admin-only, and template payloads remain excluded from participant survey responses.

## Operational Notes

- Adds a database migration for admin page-template storage.

So the practical rule is: while coding, maintain unreleased.md; when preparing a production release, promote it with release:prepare.
